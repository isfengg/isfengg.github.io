If you have already set the trace speed to the hightest level in system perferences, and you still feel the mouse move slowly. Here is a tip!

First, check your speed level in terminal.

defaults read -g com.apple.mouse.scaling

The result should be 3. And I strongly recommand you set the number between 6 and 8, or it is too fast to control. Run the code below in terminal.

defaults write -g com.apple.mouse.scaling 7

This setting will not make effect until you restart your machine.

Next time, you still can use this method to change the speed level. Or in system perferences, the highest level still is 3.

In this way, the track pad would not be affected.
